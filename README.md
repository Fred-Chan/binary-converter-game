# 二进制加密解密游戏

一个交互式的教育游戏，帮助学生学习字母、二进制数字和黑白图片之间的转换。

## 功能特点

### 📝 文本转换器
- **文字转二进制**：输入任何英文文字，立即查看它对应的二进制代码
- **二进制转文字**：输入二进制数字，还原出原始文字
- 使用8位ASCII编码，直观展示计算机如何存储数据

### 🎨 黑白图片转换
- **二进制转图片**：把二进制数字变成直观的黑白格子图案
- **文字转图片**：将英文文字转换为黑白格子可视化表示
- 白色方块 = 1，黑色方块 = 0

### 🎮 解码游戏
- **多难度选择**：
  - 简单模式：单个字母
  - 中等模式：短单词（3-5个字母）
  - 困难模式：长单词（7-10个字母）
- **计分系统**：
  - 基础分数：每题10分
  - 连胜奖励：连续答对可获得额外分数
  - 最高分记录：保存在本地存储中
- **提示系统**：遇到困难时可以获取提示

## 本地运行

1. 克隆此仓库：
```bash
git clone https://github.com/Fred-Chan/binary-converter-game.git
cd binary-converter-game
```

2. 在浏览器中打开 `index.html` 文件

不需要安装任何依赖，这是一个纯前端项目！

## 技术栈

- HTML5
- CSS3 (包含响应式设计)
- 原生 JavaScript (无需框架)

## 教育价值

这个项目旨在帮助学生：
- 理解计算机如何使用二进制存储信息
- 学习ASCII编码的基本原理
- 通过游戏化的方式练习二进制转换
- 建立数字与可视化表示之间的联系

## 部署

### Vercel 部署
1. 将项目推送到 GitHub
2. 在 [Vercel](https://vercel.com) 创建账号
3. 点击 "New Project"
4. 导入此 GitHub 仓库
5. 点击 "Deploy"

### GitHub Pages 部署
1. 进入仓库设置
2. 找到 "Pages" 选项
3. 选择 main 分支作为源
4. 保存设置
